# ערב פוקר 🂡

אפליקציה לניהול ערבי פוקר של חברים - מעקב אחרי buy-ins, ספירת ז'יטונים בסוף, וחישוב אופטימלי של ההעברות בין השחקנים.

## הרצה מקומית

```bash
npm install
npm run dev
```

## בנייה לפרודקשן

```bash
npm run build
```

## פיצ'רים

- ניהול שחקנים עם buy-in התחלתי
- הוספת buy-ins נוספים במהלך המשחק (סכום מהיר או ידני)
- "סגירה ב-0" לשחקן שיוצא באמצע ומחזיר ז'יטונים
- תמיכה בשחקני מזומן שזוכים בכסף (לא רשומים באפליקציה)
- אלגוריתם debt simplification לחישוב מינימום העברות
- שמירה אוטומטית ב-localStorage - לא מאבדים נתונים אם הדף נסגר

## טכנולוגיה

- React 18
- Vite
- Tailwind CSS
- Lucide React (icons)
